Anaconda使用

从DL目录cmd进入conda activate bicausal虚拟环境

激活虚拟环境venv\Scripts\activate

在venv中下载环境(不开梯子)

查看环境nvidia-smi

创建环境：conda create –name myenv(环境名) python=3.8

激活环境：conda activate myenv

退出环境：deactivate

查看所有环境：conda env list

删除环境：conda env remove –name myenv

安装包：conda install 包名=1.2.3(版本号)

卸载包：conda remove 包名

查看已安装包：conda list

查看conda版本：conda –version
